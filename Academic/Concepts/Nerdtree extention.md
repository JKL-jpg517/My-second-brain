### . 先看懂树的“长相”

在 NERDTree 的视窗里，符号是非常直观的：

- **`▸` (向右的三角)**：代表这是一个**折叠状态**的文件夹。
    
- **`▾` (向下的三角)**：代表这是一个**展开状态**的文件夹。
    
- 没有任何箭头、前面没有任何修饰的：就是你可以编辑的**普通文件**（比如 `main.py`）。
    

---

### 2. 基础驾驶：移动与打开

把光标移动到左侧的文件树区域（如果光标不在树上，按一下你刚才设的 `Ctrl+n` 或者按 `Ctrl+w` 再按 `h` 切过去）。

- **上下移动**：用 **`j`**（下）和 **`k`**（上）。强迫自己不要碰键盘右下角的方向键！
    
- **展开/折叠文件夹**：把光标停在一个文件夹上，按下 **`Enter` (回车)** 或者 **`o`** (open)。
    
- **打开文件**：把光标停在一个文件上，按下 **`Enter`** 或者 **`o`**。文件会在右边的主窗口立刻打开。
    

---

### 3. 高阶驾驶：分屏与焦点切换 (极客必学)

这是 Vim 比 VS Code 爽很多的地方——它的分屏极度自由。

- **`i` (上下分屏打开)**：停在文件上按 `i`，右边会自动切成上下两块屏幕打开这个文件。
    
- **`s` (左右分屏打开)**：停在文件上按 `s`，右边会切成左右两块屏幕（看对比代码时简直是神技）。
    
- **如何跳出这棵树？**：当你按 `Enter` 打开文件后，你的光标可能还在左边的树上。这时候按 **`Ctrl + w` 接着按 `l`** (向右跳)，光标就会去到右边的代码区。想回树上，就按 **`Ctrl + w` 接着按 `h`** (向左跳)。
    

---

### 4. 树的管理魔法：增删改查

你不仅能“看”树，还能直接在树上“种树”和“砍树”。

- **`m` (Menu)**：把光标停在某个文件夹上，按下 `m`。底下会弹出一排菜单：
    
    - 选 `a` (add)：可以新建文件或文件夹（最后加个斜杠 `/` 就是建文件夹）。
        
    - 选 `m` (move)：可以重命名文件。
        
    - 选 `d` (delete)：可以删除文件。
        
- **`I` (大写的 i)**：一键显示/隐藏所有的隐藏文件（比如 `.git` 或 `.vimrc`）。
    
- **`C` (大写的 c)**：把当前选中的文件夹变成这棵树的**根目录**（进入子项目时非常有用）。按 `u` (up) 可以回到上一层。
    

---

### 💡 救命稻草

如果你忘了快捷键，千万别去背！在光标停留在文件树里时，按一下 **`?`**，NERDTree 会立刻在顶部弹出一张完整的快捷键“小抄”。再按一次 `?` 就能关掉。




## 挑文件夹+开新文件夹为根
### 第一招：顺藤摸瓜（通过快捷键向上游走）

仔细看你截图的左上角，有一行字写着 **`.. (up a dir)`**。在 Linux 系统里，`..` 永远代表“上一级目录”。

1. **常规走法**：把光标（用 `k` 键）移动到 `.. (up a dir)` 这一行，按下 **`Enter`**。你会发现树展开了，显示出了上一级（也就是 `~` 或 `/home/jhj/`）的内容。
    
2. **极客走法（推荐）**：不管你的光标在树的哪个位置，直接按键盘上的 **`u`** (up)。
    
    - 按一下 **`u`**，树的根节点就会立刻跳到上一级目录。你可以一直按 `u`，甚至能退到 Linux 的最底层根目录 `/`。
        
3. **锁定新地盘**：当你退到你想要的目录后，按下大写的 **`C`** (Change Root)，这个目录就会变成你这棵树的“新树根”。
    

---

### 第二招：空降神兵（直接用命令打开指定路径）

如果你现在在 `~/learning/`，但你突然想去修改系统的配置文件（比如 `/etc/` 目录下的文件），按 `u` 一层层退回去太慢了。你可以直接“空降”：

在 Vim 的普通模式下，输入冒号进入命令模式：

Vim Script

```
:NERDTree /你/想要/去的/绝对路径
```

比如，你想直接打开你的整个用户主目录（家目录），你可以输入：

Vim Script

```
:NERDTree ~
```

按下回车，NERDTree 就会立刻抛弃现在的 `learning` 文件夹，直接为你生成一棵以 `~` 为根节点的新树。

进入：
### 1. 只是想“看一眼”里面的内容（展开）

如果你只是想看看文件夹里有什么，不改变当前的整体树形结构：

- **操作**：用 **`j`**（下）或 **`k`**（上）把光标移动到你想看的文件夹上。
    
- **按键**：按下 **`Enter` (回车)** 或者小写字母 **`o`**。
    
- **效果**：前面的小三角会从 `▸` 变成 `▾`，文件夹就像手风琴一样展开了（你在截图里展开 `mnt` 和 `c` 就是用的这个效果）。再按一次就会合上。
    

---

### 2. 想彻底“进入”并把它作为工作区（更换根节点）

你现在的树太庞大了，包含了整个操作系统的文件，看起来会很乱。如果你想专门进入比如 `Users/` 这个文件夹，让它成为你当前专注的“主场”：

- **操作**：把光标停留在你想进入的文件夹上（比如 `Users/`）。
    
- **按键**：按下**大写的 `C`**（也就是 `Shift + c`，代表 Change root）。
    
- **效果**：NERDTree 会瞬间刷新，把无关的文件全部抛弃，将 `Users/` 提拔为这棵树的最顶端（根节点）。你的世界瞬间就清净了。
