# 4.1c Fundamental Decomposition: $x = x_r + x_n$

## The Decomposition

Every vector $x \in \mathbb{R}^{n}$ splits uniquely into:

$$x = x_r + x_n$$

- $x_r \in$ Row space (the "effective" part)
- $x_n \in$ Nullspace (the "ineffective" part)

## Why Each $b$ in the Column Space Comes from Exactly One $x_r$

Apply $A$ to the decomposition:

$$Ax = A(x_r + x_n) = Ax_r + 0 = Ax_r$$

- $x_n$ is killed by $A$ — the ineffective part
- $x_r$ is what actually produces output — the effective part

This is like decomposing a force into components: only the component along the direction of action matters.

**There is a bijection between the row space and the column space.** Each $b$ in $C(A)$ corresponds to exactly one $x_r$ in $C(A^{T})$.

## The Hidden Invertible $r \times r$ Matrix

Inside $A$ there is an $r \times r$ invertible matrix that maps $x_r \to b$.

The mapping $x_r \mapsto Ax_r$ is a bijection → invertible.

When we include the nullspace (all $x$ where $Ax = 0$), the overall $A$ becomes non-invertible — but the restriction to the row space is.

## The Other Side

The same decomposition holds in $\mathbb{R}^{m}$: any $y = y_r + y_n$, where $y_r \in$ Column space and $y_n \in$ Left nullspace.
