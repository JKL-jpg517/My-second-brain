### ⚖️ 终极执行铁律（Survival Guide）

1. **周度复盘与防守熔断：** 每周日晚审查下周校内课表。如果下周有核心专业课的期中考，**毫不犹豫地将当周的 CSdiy 预算砍到 5 小时以下**。降转之后的新绩点是你去密院或北美的绝对门票，任何时候 **GPA > 一切**。
    
2. **大段连续时间神圣不可侵犯：** 无论是写 CS106B 的递归还是 CSAPP 的 C 语言指针，都必须保证拥有 3 小时以上不被打断的时间块。永远不要试图在两个课间的 20 分钟里去 Debug 一个死锁。
    
3. **博客即资产（Blog as Asset）：** 大牛的代码从不只留在硬盘里。在 Phase 0 掌握 GitHub 后，立刻用静态框架（如 Hexo/Hugo）搭建个人博客。把你证出的离散数学公式、写出的 Raft 选举逻辑用中英双语记录下来。大二时，这个带有时间戳的技术博客，将是你简历上最锋利的武器。
