### 🧠 核心心法：Vim 的“主谓宾”语法

不要死记硬背长串指令，记住公式：**操作（动词） + 次数 + 范围（名词）**

- **动词**：`d` (删除), `c` (修改), `y` (复制), `v` (选中)
    
- **名词**：`w` (单词), `$` (行尾), `G` (文档末尾), `i"` (引号内部)
    
- _例：`d3w` = 删掉后面 3 个单词。`ci"` = 修改引号里的内容。_
    

### 1. 🚪 进出模式 (Mode Switching)

_你永远的避风港是 Normal（普通）模式。_

|**指令**|**动作**|**极客使用场景**|
|---|---|---|
|**`jj`**|**退回 Normal 模式**|你自己配置的绝对核心，打完字立刻按。|
|**`i`**|在光标**前**插入|最基础的打字入口。|
|**`a`**|在光标**后**插入|光标在单词末尾，想继续往后写时用。|
|**`A`**|跳到**行尾**并插入|想在这一行最后加个标点或注释。|
|**`I`**|跳到**行首**并插入|想在这一行最前面加个重点符号。|
|**`o`**|在**下方**新开一行插入|写完这句，想另起一行继续写。|
|**`O`**|在**上方**新开一行插入|发现上面漏了前提条件，补一行。|

### 2. 🧭 光标移动 (Navigation)

_彻底抛弃鼠标和方向键。_

| **指令**              | **动作**       | **极客使用场景**         |
| ------------------- | ------------ | ------------------ |
| **`h` `j` `k` `l`** | **左、下、上、右**  | 基础移动，右手食指长驻 `j`。   |
| **`w`**             | 跳到**下个**单词开头 | (Word) 在句子里快速横移。   |
| **`b`**             | 跳到**上个**单词开头 | (Back) 往回找错别字。     |
| **`e`**             | 跳到当前单词**末尾** | (End) 准备在词尾加个 's'。 |
| **`0`** (零)         | 跳到绝对行首       | 回到最左边。             |
| **`^`**             | 跳到行首第一个非空格字符 | 写代码时跳到有内容的开头。      |
| **`$`**             | 跳到绝对行尾       | 去看这一行最后写了啥。        |
| **`gg`**            | 跳到文档**第一行**  | 回到文章标题处。           |
| **`G`**             | 跳到文档**最后一行** | 去底部写新的日记。          |

### 3. 🗡️ 增删改查 (Editing)

_外科手术刀级别的精准操作。_

|**指令**|**动作**|**极客使用场景**|
|---|---|---|
|**`x`**|删掉光标下的**一个字符**|相当于键盘上的 Delete 键。|
|**`dd`**|删掉（剪切）**整行**|这行逻辑写废了，整行端掉。|
|**`cc`**|删掉整行，并**进入插入**|这行全错，我要直接重新写。|
|**`D`**|删掉光标到**行尾**的内容|后半句全是废话，直接砍掉。|
|**`C`**|删掉光标到行尾，并插入|后半句要重写。|
|**`r`**|**替换**当前字符|(Replace) 把 `a` 换成 `b`，按 `rb`，停留在普通模式。|
|**`/关键词`**|**搜索**关键词|敲回车后找目标。按 `n` 找下一个，按 `N` 找上一个。|

### 4. 📋 复制粘贴 (Copy & Paste)

_在 Vim 里，复制叫 Yank（拽）。_

|**指令**|**动作**|**极客使用场景**|
|---|---|---|
|**`yy`**|复制**整行**|复制这段代码/笔记框架。|
|**`p`**|在光标**下方**粘贴|(Paste) 把刚才复制的/删掉的东西贴出来。|
|**`P`**|在光标**上方**粘贴|贴在上面。|
|**`v`**|进入**视觉选择**模式|(Visual) 像鼠标拖拽一样，用 `hjkl` 移动高亮选中，然后按 `y` 复制或 `d` 删除。|
|**`V`**|选中**整行**|按行选取多行代码。|

### 5. 🛟 救命指令 (Safety Net)

_胆子大一点，反正随时可以后悔。_

|**指令**|**动作**|**极客使用场景**|
|---|---|---|
|**`u`**|**撤销** (Undo)|删错了！按一下立刻恢复。|
|**`Ctrl + r`**|**重做** (Redo)|撤销过头了，再变回来。|
|**`<CR><CR>`**|取消搜索高亮|搜完东西满屏黄底，连按两次回车清空（你刚才配好的神技）。|

### ⚡ 组合神技推荐 (The "Wow" Factor)

这几个组合键是你拉开效率差距的绝杀：

1. **`ciw`** (Change Inner Word)：光标在某个单词中间时，按下直接清空该单词并开始打字。（改错别字神器）
    
2. **`ci"`** 或 **`ci(`**：清空引号/括号里的内容并开始打字。（写 Python 传参时极其舒适）
    
3. **`d}`**：删除从当前光标到这个段落末尾的所有内容。
    
4. **`dt.`** (Delete To .)：删除从当前光标一直到本句的句号（英文句号）
